    Изучение дисциплины “Базы данных” ставит своей целью изучения теории проектирования реляционных баз данных, в том числе распределенных, а в практической части - освоение основ языка структурированных запросов к базам данных SQL, а также языка обработки гипертекстов PHP, позволяющего, в частности, создавать веб-сайты с использованием запросов к базе данных. В курсе рассматриваются основные понятия языка SQL: запросы, операторы, функции, а также языка PHP: переменные, операторы, функции, массивы, работа с формами и базами данных.

    В процессе освоения дисциплины студент должен получить основные навыки написания запросов к базам данных с использованием языка SQL, а также создания гипертекстовых страниц, выполняющих обработку сценариев: отправку на сервер форм, обращение к базе данных MySQL и т.п.

    Предполагается, что студент имеет навыки работы в текстовых редакторах, владеет языком HTML в объеме курса "Интернет-технологии" и основами языка C.

    Создание веб-сайтов, страницы которых генерируются посредством запросов к базе данных, является непременной составляющей квалификации любого специалиста в области информационных технологий. Поэтому рекомендуется отнестись к изучению данного предмета со всей ответственностью и серьезностью.

    В процессе освоения данной дисциплины студенту необходимо:

    1. изучить конспекты лекций (общая теория в части 1 и основы языка PHP в части 2), выполняя приведенные в тексте лекций упражнения и примеры листингов программ (как в тексте лекция, так и в приложении);
    2. выполнить лабораторные работы (т.е. написать программы в соответствии с заданием), результаты оформить в виде отчетов;
    3. выполнить контрольную работу;
    4. сдать экзамен/зачет.

    Для выполнения заданий необходима установка программного обеспечения - комплект Denwer с сайта http://www.denwer.ru/ в соответствии с инструкциями на указанном сайте.


    Внимательно прочтите следующие рекомендации:

    1. При изучении курса “Базы данных” помимо конспекта лекций целесообразно использовать источники, указанные в списке литературы. При этом могут изучаться и другие источники аналогичной тематики, не вошедшие в данный список.
    2. Краткая инструкция по выполнению запросов SQL содержится в начале теоретической части лабораторной работы N 1, по запуску программ (скриптов) PHP - в начале теоретической части лабораторной работы N 2..
    3. Контрольная работа и отчеты по лабораторным работам должны быть высланы со страниц нашего сайта. С порядком отправки выполненных работ можно ознакомиться в инструкции по отправке, доступной по ссылке.
    4. После проверки работы, а также отчётов по лабораторным работам, преподавателем Вы получите рецензию с оценкой и возможными замечаниями по представленному решению для исправления или доработки. В этом случае после доработки Вы должны повторно выслать контрольную работу, отчёт по лабораторной работе на проверку.
    5. Экзаменационная оценка выставляется по итогам выполнения всех лабораторных работ, контрольной работы и экзаменационного задания.

    Получить консультацию преподавателя можно по контактным данным (см. раздел "Консультации ").
